### 와치테이블 소개
>**"음식을 사랑하고 효율을 중시하는 한국사람들! <br>
맛집 앞에서 끝도 없는 웨이팅도 불사하지만 조금 더 효율적인 방법으로 맛집 탐방을 할 순 없을까?!"** <br><br>
가고싶은 가게의 상황을 미리 알 수 있으면 어떨까요? 와치테이블은 말 그대로 보이는 테이블이란 뜻으로 내가 가고자 하는 매장의 혼잡도를 보여줍니다.
